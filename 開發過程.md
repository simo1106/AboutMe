```const searchableItems = document.querySelectorAll('.searchable-item');

searchInput.addEventListener('keyup', function (event) {
  const filter = event.target.value.toUpperCase();

  searchableItems.forEach(function (item) {
    const text = item.innerText.toUpperCase();

    if (text.includes(filter)) {
      item.style.display = '';
    } else {
      item.style.display = 'none';
    }
  });
});
```

上述功能是前段動態呼叫使用，所以它不是 Elasticsearch 搜尋。
要讓 index.html 用 Elasticsearch，你需要：
- Elasticsearch container in docker 正在跑
- npm start 讓 server.js 跑在 localhost:3000
- search.js 改成呼叫 /api/search?q=關鍵字
- Elasticsearch 裡要先有你網站內容的資料

後端 Elasticsearch 已經能用，但前端 index.html 還沒接上 Elasticsearch。

---
**Elasticsearch 該如何擁有我的index.html網站資料？**

把每個 .searchable-item 區塊轉成 Elasticsearch 文件。這樣你不用在 Postman 手動一筆一筆塞網站內容。
